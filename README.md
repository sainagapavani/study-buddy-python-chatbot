# 🤖 Study Buddy - Python Chatbot

## 📖 About

Study Buddy is a Python console chatbot designed to help beginners learn Python programming. It answers predefined Python-related questions, provides clean explanations, and offers an interactive command-line learning experience.

---

## ✨ Features

- 👋 Time-based greetings
- 💬 Interactive console chatbot
- 📚 Covers Python basics and advanced topics
- 🧹 Cleans user input automatically
- 🛡 Handles invalid and unknown questions gracefully
- ⌨ Multiple exit commands (`exit`, `quit`, `bye`, `goodbye`)
- 🎯 Typewriter-style response effect
- ⚡ Keyboard Interrupt (Ctrl+C) handling

## ▶ How to Run

1. Clone the repository

```

```

2. Navigate to the project folder

```
cd study-buddy-python-chatbot
```

3. Run the program

```
python python_chatbot.py
```
